# Aparência do Checkout

As páginas de Carrinho de Compras, Checkout e Nova Central (do cliente) não permitem edições HTML, ou seja, não é possível personalizar a estrutura de ambas as páginas, mas é possível editar as cores das mesmas pelo painel da plataforma.

Para isso acesse **Configurações > Minha loja > Aparência do carrinho**. Neste local você encontrará todas as cores passíveis de edição, e ao lado direito em "Pré-visualização" é possível ver quais cores estão sendo alteradas e como está ficando a página.
