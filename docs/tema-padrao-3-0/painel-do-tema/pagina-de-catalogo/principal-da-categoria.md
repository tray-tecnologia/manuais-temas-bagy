# Principal da Categoria

Esse bloco configura os itens do menu de filtro exibidos na loja.&#x20;

Em **Configurações da seção** temos:

* **Exibir filtro:** é possível selecionar se deseja exibir o menu de filtro na loja ou não.
* **Filtro recolhido por padrão:** é possível selecionar se deseja exibir o menu de filtro de maneira "aberta" ou "fechada". Veremos exemplos a seguir.
* **Filtro de preço De-Até:** é possível escolher se deseja exibir o filtro de preço padrão da plataforma ou onde é possível escolher entre uma faixa de preço. Também veremos exemplos a seguir.
