# Marcas

Para esse bloco é possível realizar algumas configurações, tais como:

* Selecionar se deseja exibir os itens ou não através do clique no ícone de olho
* Remover um item
* Arrastar o item para reposicioná-lo
* Inserir a Imagem, o Texto alternativo e o Link da marca.



**Imagem da marca**

Possível cadastrar uma imagem para a marca e também removê-la caso não deseje mais exibir a imagem.



**Texto alternativo**

Exibido caso a imagem da marca não tenha sido cadastrada. Se a imagem for cadastrada será exibido como o atributo _ALT_ da imagem.

::: info **Atributo ALT:**
Sua principal função é garantir a acessibilidade para utilizadores com deficiência visual, que usam leitores de tela, e para os motores de busca, como o Google, que o utilizam para entender o conteúdo e o contexto da imagem para otimização de SEO
:::


**Link**

Deve ser cadastrado para que se tenha acesso a página da marca e os seus produtos.

::: info Mais
Para cadastrar as marcas no painel da plataforma e ter o seu link acesse: **Produtos > Marcas e Fabricantes.** Para maiores detalhes [clique aqui](https://basedeconhecimento.bagy.com.br/hc/pt-br/articles/44774623772180-Cadastrando-marcas-e-fabricantes-na-sua-loja-Bagy-3-0).
:::

Lembrando que é possível cadastrar no **máximo 10** marcas.
