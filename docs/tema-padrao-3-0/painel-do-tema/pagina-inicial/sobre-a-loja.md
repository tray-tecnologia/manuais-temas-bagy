# Sobre a Loja

Esse grupo contém as configurações da exibição do Sobre a Loja na página inicial.

Sendo eles:

* **Imagem**: possível cadastrar uma imagem para esse bloco. Recomendamos tamanho de 525x405px. Lembrando que se deixar vazio não será exibido o bloco.
* **Título:** edita o titulo do bloco. Lembrando que se deixar vazio não será exibido o título.
* **Descrição:** edita a descrição do bloco. Lembrando que se deixar vazio, a mesma não será exibida.
