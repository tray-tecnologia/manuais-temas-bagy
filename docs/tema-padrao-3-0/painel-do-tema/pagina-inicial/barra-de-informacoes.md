# Barra de Informações

Para esse bloco é possível realizar algumas configurações, tais como:

* Selecionar se deseja exibir os itens ou não através do clique no ícone de olho
* Inserir o titulo e a descrição
* Arrastar os blocos para reposiciona-los

![image](/tema-padrao-3-0/painel-do-tema-pagina-inicial-barra-informacao.png)

Temos quatro itens com ícones pre-definidos:

* **Ícone cartão**
* **Ícone caminhão**
* **Ícone devolução**
* **Ícone cadeado**
