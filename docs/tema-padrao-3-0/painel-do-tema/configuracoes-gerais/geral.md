# Geral

Neste item é possível definir algumas configurações gerais da loja, tais como:


**Raio das bordas**

Possível escolher como será o estilo dos botões, barra de busca e campos de input da sua loja, sendo eles:

* Arredondado (Padrão do tema)
* Pouco arredondado
* Quadrado
