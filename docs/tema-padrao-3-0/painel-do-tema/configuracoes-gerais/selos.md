# Selos

Neste item é possível setar algumas configurações para os selos que serão exibidos na loja, tanto nas listagens quanto na pagina do produto.

**Configurações**:

* **Mostrar selos:** Ao habilitar essa configuração os selos serão exibidos na loja.
* **Usar selos com texto:** Os selos serão exibidos como texto e cor de fundo e não como imagem.&#x20;

::: info Lembrando que:
Ao deixar a configuração **Usar selos com texto** desabilitada, os selos serão exibidos como imagem na loja e para inserir/alterar essas imagens é através do item [Imagens Padrões](../../configuracoes/imagens-padroes)
:::

Ao habilitar a configuração **Usar selos com texto**, você poderá inserir os textos exibidos para cada selo:

* **Selo produto novo:** para os produtos novos, em lançamento.
* **Selo produto em destaque:** para os produtos que queira destacar na loja.
* **Selo frete grátis:** para os produtos que oferecerão frete grátis.
* **Selo adicional:** utilizado para uma ocasião especial, como a Black Friday ou alguma promoção.
