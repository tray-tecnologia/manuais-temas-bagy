# Imagens na listagem

**Formato da imagem do produto**

Possível escolher como será a exibição da imagem do produto nas vitrines, sendo elas:

* Quadrada
* Retrato
* Paisagem

![image](/tema-padrao-3-0/painel-tema-configuracoes-geral-imagem-listagem.png)


#### **Quadrada**

A imagem será exibida na proporção 1:1

Por exemplo, a imagem deve ter o tamanho 1000 x 1000 px para ser exibida corretamente

![image](/tema-padrao-3-0/painel-tema-configuracoes-geral-imagem-listagem-2.png)


#### **Retrato**

A imagem será exibida na proporção 3:4

Por exemplo, a imagem deve ter o tamanho 1000 x 1333 px para ser exibida corretamente

![image](/tema-padrao-3-0/painel-tema-configuracoes-geral-imagem-listagem-3.png)


#### **Paisagem**

A imagem será exibida na proporção 16x9

Por exemplo, a imagem deve ter o tamanho 1000 x 563 px para ser exibida corretamente

![image](/tema-padrao-3-0/painel-tema-configuracoes-geral-imagem-listagem-4.png)
