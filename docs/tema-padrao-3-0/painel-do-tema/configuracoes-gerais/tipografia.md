# Tipografia

Neste item é possível selecionar qual fonte deseja utilizar em sua loja. Lembrando que a mesma fonte será aplicada em todos os itens.

É possível escolher entre três opções:

* Outfit (Padrão do tema)
* Inter
* Lora
